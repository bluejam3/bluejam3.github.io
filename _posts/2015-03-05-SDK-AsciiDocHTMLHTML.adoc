==== Twitter製のAsciiDoc記法で作るHTMLスライドツール。
---
*CDKの使い方*

===================================

*CDKのインストール方法１*

---

pipで行う方法

----
$ pip install cdk
----


*CDKのインストール方法2*

---

クイックインストール
----
$ sudo easy_install cdk
----

===================================


===================================

*サンプルデッキをデッキを生成する*

---

----
$ cdk --generate=sample.asc
----

===================================

===================================


*スライドを制作する。* 

---

ブラウザーで開くためにHTMLを生成する。
----
$ cdk -o sample.asc
----

===================================
